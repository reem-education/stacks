
This package makes use of ViSP and KDL.


ViSP is an open-source with a GNU GPLv2 license developed and maintained by the INRIA Lagadic team located at INRIA Rennes - Bretagne Atlantique. For more information please check http://www.irisa.fr/lagadic/visp/visp.html. 

ViSP, standing for Visual Servoing Platform, is unique. This software is a complete cross-platform library that allows prototyping and developing applications in visual tracking and visual servoing.


KDL stands for Kinematics and Dynamics Library and it is a sub-project of Orocos.
